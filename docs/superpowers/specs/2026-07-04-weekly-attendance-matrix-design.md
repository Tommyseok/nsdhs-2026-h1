# 전체출석현황 주차별(날짜별) 매트릭스 — 설계

> 작성 2026-07-04 · 승인됨

## 목적
출석을 %와 최근 4주 점뿐 아니라 **주일 날짜별로** 파악. 전체 + 각 반별 모두.

## 결정
- 방식: **날짜별 매트릭스 표** (행=학생, 열=주일 날짜)
- 기간: 기본 **2학기(2026-06-07~)** + `1학기 포함` 토글(1/11~)
- 적용: **attendance-overview.html만** (셀별 그룹 뷰가 반별 현황 커버)

## 변경 사항 (attendance-overview.html 1개)
1. **필터 바**: `[📊 요약] [📅 주차별]` 보기 토글 + 주차별일 때만 `☑ 1학기 포함` 체크박스.
   정렬·필터 드롭다운은 요약 전용(주차별에선 숨김), 검색은 양쪽 적용.
2. **JS**:
   - `VIEW_MODE`('summary'|'matrix'), `INCLUDE_H1`(bool), `H2_START='2026-06-07'`
   - `matrixWeeks()`: 시작일(토글에 따라 1/11 or 6/7)부터 이번 주 일요일까지 매주 (미래 주 제외)
   - `statusFor(name, week, data)`: 사용자 입력(Supabase 병합) 우선 → 1학기 historical → '·'
   - `renderMatrix()`: 대가족 → 셀 카드(보고 싶은 친구들 포함, `🏠` 배지 유지).
     카드=표: thead 날짜(M/D), tbody 학생행(●◐○·, 끝에 누적%), tfoot 주차별 출석수/재적
3. **CSS**: 매트릭스 표(가로 스크롤 `overflow-x`, 이름 열 sticky, 상태색 기존 재사용)

## 데이터
변경 없음 — 기존 `userAtt()`(Supabase attendance + localStorage 병합) + `HISTORICAL_ATTENDANCE` 재사용.

## 검증
브라우저: ① 토글 전환 ② 2학기 주차 열(6/7~현재) ③ 1학기 포함 시 1/11~ ④ 상태·누적%가 요약 보기와 일치 ⑤ 검색 동작 ⑥ 모바일 폭 가로 스크롤 ⑦ 콘솔 에러 0.
